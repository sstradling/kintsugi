# kintsugi

A zen 3D shatter-and-mend puzzle game for iOS and Android. Players reassemble
shattered 3D objects, then bind the seams with a chosen filler — gold,
platinum, lava, wood, bioluminescent gel, rainbow glitter, sunlight, moss.

No timers. No fail states. A personal museum of mended things.
